# XSES-FSI: towards eXtreme scale Semi-Structured discretizations for Fluid-Structure Interaction

## Main repository (WIP)

To download the software stack use the following command

```bash
git clone --recurse-submodules https://github.com/xsesfsi/xsesfsi.git
```
